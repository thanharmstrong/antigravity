# Shadcn UI Reference

## Common Components

| Component | Command | Usage |
|-----------|---------|-------|
| Button    | `add button` | Primary actions |
| Card      | `add card` | Content containers |
| Input     | `add input` | Form fields |
| Dialog    | `add dialog` | Modals and overlays |
| Tabs      | `add tabs` | Section switching |
| Select    | `add select` | Dropdown menus |
| Popover   | `add popover` | Contextual menus |

## Theming Variables
Modify these in `src/index.css`:
- `--background`: Page background color
- `--foreground`: Major text color
- `--primary`: Main accent color
- `--muted`: De-emphasized text/background
- `--accent`: Hover/Active states
- `--radius`: Global border radius (e.g., 0.5rem)
